# Caio Durazzo

Site oficial de Caio Durazzo desenvolvido em React + Vite, com direção visual editorial, páginas institucionais, discografia, agenda, área de staff e página dedicada do projeto Crazy Legs.

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Estrutura

- `src/components`: seções, páginas e painéis
- `src/data`: textos, links e dados editoriais
- `src/context`: estado local para agenda, vídeos, hero e discografia
- `assets`: imagens e materiais visuais do projeto

## Observações

- O painel `/staff` usa persistência local no navegador para edição de conteúdo.
- A página `Crazy Legs` possui hero dedicada, seção institucional, colagem de arquivo, discografia filtrada e fechamento editorial.
